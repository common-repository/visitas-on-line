== Description ==

Make a record of all the pages requested by an IP, also attaching Country, time of registration and where it came from.

== Installation ==

1. Upload 'On-line Visits' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Done!
